# Kohli by Numbers — A Data-Driven Verdict

## Overview
An end-to-end data analytics project analyzing 15 years 
of Virat Kohli's international cricket career across 
532 innings (ODI, T20I, Test) from 2008 to 2023.

## Key Findings
- Peak ODI year: 2018 (average: 85.86)
- Worst ODI year: 2022 (average: 21.88)
- Best opponent: New Zealand (average: 53.00)
- 76 centuries across all formats
- ODI is his strongest format (avg: 48.79)

## Tools Used
- Python (Pandas) — data cleaning & analysis
- Power BI — interactive dashboard
- CSV — cleaned dataset

## Files
- analysis.py — Python cleaning and analysis script
- kohli_clean.csv — cleaned dataset (all 3 formats)

## Dashboard
Built using Power BI with navy/gold theme showing:
- Average runs by format
- Year-wise career journey
- Performance vs each opponent
- Innings distribution by format
